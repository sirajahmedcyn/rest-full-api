# Student Management System

A Node.js-based student management system with Express.js, MongoDB, and EJS templating.

## Features

- User authentication and authorization
- Admin dashboard
- Blog functionality
- File uploads

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/sirajahmedcyn/rest-full-api.git
   cd rest-full-api
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file with your MongoDB connection string and other secrets.

4. Start the server:
   ```
   npm start
   ```

The application will run on `http://localhost:3000` (or the port specified in your config).

## Technologies Used

- Node.js
- Express.js
- MongoDB with Mongoose
- EJS for templating
- Multer for file uploads
- bcryptjs for password hashing
- Express-session for session management
